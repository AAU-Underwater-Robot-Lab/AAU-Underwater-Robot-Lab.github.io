Underwater Technologies Group

Offshore drones, ROVs, UUVs, and marine autonomy

Our Mission

We develop intelligent robotic platforms equipped with advanced perception, navigation, and control systems to support inspection, maintenance, and research in challenging offshore and subsea environments.

Capabilities

Robotics & Autonomy

Development of ROVs, UUVs, and UAVs for subsea operations

Perception & Sensor Fusion

Integration of sonar, vision, ML-based perception systems

Modelling & Control

Hydrodynamic modelling, control algorithms, simulation

System Integration

Hardware-software integration, field deployments

Public Repositories

  • dataset_browser — Public catalog of synthetic underwater datasets.
  • cuvis.ros — CUVIS driver for hyperspectral camera.
  • ros2_ws — Workspace container for ROS 2-based development.
  • waterlinked_3d_sonar_driver — ROS2 code for Water Linked Sonar 3D-15 integration.
  • liboculus — ROS2-compatible version of liboculus library.
  • BlueROS2 — Temporary repository for BlueROV2 integration under ROS2.
  • blender_virtual_underwater — Blender-based virtual underwater environment simulation.
  • acomar — ACOMAR software repository for ROS integration.
  • create3_docs — Documentation for Create® 3 robot, archived for reference.
  • synthetic_fls_dataset — Dataset for synthetic sonar generation using Blender.
  • cubert_fuchsia_driver — Development driver for Cubert Fuchsia hyperspectral system.
  • ROS_BlueROV2 — ROS stack tailored for BlueROV2 operations.
  • optical_acoustic_sensors — Sensor integration library for acoustic and optical data fusion.
  • optical_acoustic_sensors_gui — GUI frontend for hybrid acoustic-optical sensor systems.
  • BlueROV2_Web_Interface — Web-based interface for controlling BlueROV2 (WIP).
  • BlueROV2_Simulink_Simulator — Simulink-based simulator for BlueROV2 vehicle dynamics.
  • bluerov2_simulator — Scripts for integrating BlueROV2 with UUV Simulator and ROS.
